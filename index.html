Product Board Timeline

Connect your board

Enter your Asana Personal Access Token, then paste any Product Board URL to generate a unified timeline view with rich milestone data.

Read-only & private. Your token is used only to make direct calls to Asana's API from your browser. It is never stored or transmitted anywhere else, and is cleared when you close the tab. This tool cannot modify any Asana data.
See the instructions below for how to generate this.
Paste any view URL from your Product Board — the project ID is extracted automatically.

How to get your Personal Access Token

1
Open Asana's developer settings Go to app.asana.com/0/my-apps — you must be logged in to Asana first.
2
Create a new token Scroll to the Personal access tokens section and click Create new token.
3
Name it and confirm Give it any name (e.g. Timeline viewer), accept the terms, and click Create token.
4
Copy the token immediately Asana will only show the token once. Copy it and paste it into the field above. It looks like 1/123456789:abcdef1234567890…
5
Get the board URL Open the Asana Product Board you want to view, copy the URL from your browser's address bar, and paste it into the Board URL field above.
⚠ Treat your token like a password. Anyone with it can read your Asana data. If you ever share a device or suspect the token is compromised, you can revoke it at any time from the same My Apps page.
